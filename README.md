**QuickShow** is a modern, high-performance movie ticket booking application built using the **MERN Stack**. It offers a seamless, real-time experience for movie enthusiasts and a comprehensive management suite for cinema administrators.

---

## ğŸš€ Key Features

### For Users
* **Secure Authentication**: Managed by **Clerk** (supports Email, Social Logins, and Phone).
* **Interactive Seat Selection**: Real-time seat maps with category-based pricing.
* **Secure Payments**: Fully integrated with **Stripe** for safe credit/debit card transactions.
* **Booking History**: Personalized dashboard to view and manage your tickets.
* **Smart Notifications**: Automated email confirmations and show reminders powered by **Nodemailer**.
* **Auto-Release Seats**: Temporary 10-minute seat locking during the checkout process to prevent overbooking.

### For Admins
* **Analytics Dashboard**: Visual data insights into revenue, active shows, and booking trends.
* **Movie Management**: Tools to add, edit, or remove movies with metadata and poster uploads.
* **Show Scheduling**: Flexible scheduling for multiple screens and timings.
* **Automation**: Background jobs handled by **Inngest** for database cleanup and reminder scheduling.
